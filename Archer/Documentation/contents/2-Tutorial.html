Simple tutorials to demonstrate the basic uses of Archer.

Drag and drop item(s) into Archer to create arrow(s)

  1. Simply drag a file or a folder or a string into Archer's main window.
  2. Change the Name, such as "item", then press Enter or TAB.

Quick way to open web search engine with key words

  1. Open the Settings window.
  2. Add an item, input following information,
    Name:b
    Cmd :http://www.bing.com/search?q=<Keyword/>&first=
  3. Save settings(Ctrl + S).
  4. Input "b" in Archer's left input box, then press Space or TAB, then input "sky 20", then press Enter.
    Archer will open website Bing.com and search keyword "sky" and show up the 20th result.

Directly search selected text

  1. Open the Settings window, add an item,
    Name:s
    Type:t
    Cmd :http://www.bing.com/search?q=
    Key :Win + S
  2. Ctrl + S save settings.
  3. Select any text in any application, then press "Win + S".
    Archer will open website Bing.com and search the selected keyword.

Quick way to operate file system object

  1. Add an item, input following information and save settings,
    Name:f
    Cmd :explorer.exe /select,%windir%\
    Tag :This is comment
  2. Win + A, input the "f", then press Space or TAB, then input "Fonts", then press Enter. Archer will open system folder and just select the directory "Fonts".

Run script

  1. Add an item, input following information and save settings,
    Name:test
    Type:bat
    Cmd :@echo"Hello %1"& pause
    Arg :World
    Tag :Select folder
    Key :Win + H
  2. Press "Win + H".This is a more complex usage of Archer, it will run the script and print "Hello World" in the console window.
    Windows batch or vbs is easy to access, Python and other script types are powerful.

Run internal C# script

  1. Add an item, input following information and save settings,
    Name:CSharp
    Type:c#
    Cmd :Clipboard.SetText("Hello World!");
    Tag :Copy string to clipboard
  2. Run this arrow, it will copy the text to clipboard.
